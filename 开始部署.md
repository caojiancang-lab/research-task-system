## ⚡ 最快部署方式（推荐）

### Windows 用户 - 3步完成

#### 第1步：双击运行部署脚本

找到项目文件夹中的 **`deploy-github.bat`** 文件，双击运行。

#### 第2步：按提示操作

脚本会自动引导您完成：
- ✅ 检查 Git 安装
- ✅ 初始化仓库
- ✅ 配置用户信息
- ✅ 提交代码
- ✅ 推送到 GitHub

**您只需要**：
1. 输入 GitHub 用户名
2. 输入邮箱
3. 先去 GitHub 创建仓库（脚本会告诉您怎么做）
4. 输入仓库地址

#### 第3步：启用 GitHub Pages

1. 访问您的 GitHub 仓库
2. 点击 `Settings` → `Pages`
3. Source 选择 `main` 分支
4. 点击 `Save`
5. 等待 1-2 分钟

**完成！** 🎉

## 🐛 遇到问题？

### 快速解决

1. **查看文档**：`GITHUB_快速指南.md` 的常见问题部分
2. **检查清单**：`DEPLOYMENT_SUMMARY.md` 的检查清单
3. **重新部署**：删除 `.git` 文件夹，重新运行脚本

### 常见问题

**Q: Git 未安装？**
- 下载：https://git-scm.com/download/win
- 安装后重启命令行

**Q: 推送失败？**
- 使用 Personal Access Token
- 查看 `GITHUB_快速指南.md` 的详细说明

**Q: Pages 404？**
- 等待 2-3 分钟
- 清除浏览器缓存
- 检查 Pages 设置
