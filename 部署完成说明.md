## 🚀 现在开始部署！

### 方式1：最简单（推荐新手）⭐⭐⭐⭐⭐

**只需3步**：

1. **双击运行** `deploy-github.bat`
2. **按提示操作**（输入用户名、邮箱、仓库地址）
3. **启用 GitHub Pages**（Settings → Pages → Save）

**时间**：5-10分钟
**难度**：⭐（非常简单）

### 方式2：手动部署

**查看文档**：`GITHUB_快速指南.md`

**步骤**：
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

**时间**：10-15分钟
**难度**：⭐⭐（需要基本命令行知识）
