### 4️⃣ 科研进展周计划系统 ⭐
**本次更新的重点功能！**

#### 核心特点

✅ **三周联动**：上周、本周、下周完美衔接  
✅ **自动计算**：系统自动识别周一到周日  
✅ **权限控制**：用户填写，管理员审核  
✅ **审核锁定**：审核后无法修改  
✅ **历史追溯**：记录所有填写和修改

#### 使用流程

**📝 用户填写计划**：
```
1. 打开科研进展任务详情
2. 点击 📅 周计划 按钮
3. 查看上周计划（参考）
4. 填写本周计划
5. 填写下周计划
6. 点击"保存计划"
```

**✅ 管理员审核**：
```
1. 打开任务详情
2. 点击 📅 周计划 按钮
3. 查看用户提交的计划
4. 点击"✅ 审核通过"
5. 审核后用户无法修改
```

#### 时间衔接示例

假设今天是 **2026年1月21日（周二）**：

```
📋 上周计划：2026-01-13（周一）至 2026-01-19（周日）
   └─ 只读，用于参考

✍️ 本周计划：2026-01-20（周一）至 2026-01-26（周日）
   └─ 可编辑（审核前）

🎯 下周计划：2026-01-27（周一）至 2026-02-02（周日）
   └─ 可编辑（审核前）
```

#### 审核机制

**审核前**：
- ✅ 用户可随时修改
- ✅ 可多次保存更新
- ✅ 管理员可查看

**审核后**：
- ❌ 用户无法修改
- ✅ 显示"已审核"标记
- ⚠️ 如需修改需联系管理员

## 💬 常见问题

**Q1：为什么新用户无法创建任务？**  
A：V2.7版本调整了默认权限，新用户默认只有查看权限。管理员需要手动授予其他权限。

**Q2：一键完成和正常完成有什么区别？**  
A：一键完成是管理员的快捷操作，直接将任务设为完成。正常流程是逐步更新进度到100%，然后审核通过。

**Q3：周计划审核后能否修改？**  
A：普通用户审核后无法修改，但管理员仍可以修改。如需修改请联系管理员。

**Q4：周计划必须每周填写吗？**  
A：建议每周填写，但不是强制的。系统会自动显示上周、本周、下周的计划，便于连续管理。

**Q5：进度更新的说明必须填写吗？**  
A：不是必须的，但强烈建议填写。进度说明有助于追溯任务进展和问题排查。
