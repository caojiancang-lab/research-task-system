# 科研任务管理系统 V3.0 - 快速上手

## 🎯 目标

让每个人都可以通过网址访问、注册、登录和使用系统。

## ⚡ 最快部署方式（推荐）

### 使用 Vercel（5分钟完成）

#### 第1步：注册账号（1分钟）

访问 https://vercel.com ，使用GitHub账号登录

#### 第2步：上传代码（2分钟）

```bash
# 如果还没有Git仓库
git init
git add .
git commit -m "Deploy to Vercel"

# 推送到GitHub
git remote add origin https://github.com/YOUR_USERNAME/research-tasks.git
git push -u origin main
```

#### 第3步：部署（2分钟）

1. 在 Vercel 点击 "New Project"
2. 选择你的 GitHub 仓库
3. 点击 "Deploy"
4. 等待部署完成

#### 第4步：获取网址

```
https://research-tasks.vercel.app
```

**分享给团队成员，开始使用！**

## 📱 使用说明

### 用户注册

1. 访问系统网址
2. 点击"注册"按钮
3. 填写信息：
   - 学号
   - 姓名
   - 密码
   - 年级
   - 学历层次
   - 学院
   - 手机号
   - 邮箱
4. 点击"注册"完成

### 用户登录

1. 输入学号
2. 输入密码
3. 点击"登录"

### 创建科研进展

1. 登录后进入"任务管理"
2. 点击"创建任务"
3. 选择"科研进展"类型
4. 填写任务信息
5. 保存

### 提交周计划

1. 进入任务详情
2. 填写：
   - 上周完成内容
   - 本周计划内容
   - 下周计划内容
3. 点击"保存计划"
4. 计划自动提交，等待审核

### 管理员审核

1. 管理员登录
2. 进入"科研进展"页面
3. 查看所有用户提交的计划
4. 点击"审核"批准计划

## 🔧 高级部署（需要数据库）

### 如果需要真正的多用户系统

#### 方案A：使用 Railway（推荐）

**优势**：自动配置数据库，一键部署

1. 访问 https://railway.app
2. 连接 GitHub 仓库
3. 自动部署（包含数据库）
4. 获取访问地址

**费用**：$5/月（包含数据库）

#### 方案B：使用 Vercel + PlanetScale

**优势**：完全免费

1. **创建数据库**
   - 访问 https://planetscale.com
   - 创建数据库 "research-tasks"
   - 导入 `server/database.sql`

2. **部署后端**
   - 在 Vercel 部署 `server` 目录
   - 配置环境变量（数据库连接）

3. **部署前端**
   - 在 Vercel 部署 `web` 目录
   - 配置 API 地址

4. **完成**
   - 获取访问地址

## 🎓 适合不同场景

### 场景1：小团队测试（<10人）

**方案**：直接部署到 Vercel
- 使用 localStorage 存储数据
- 每个人独立使用
- 无需数据库

**步骤**：
1. 上传代码到 GitHub
2. Vercel 一键部署
3. 分享网址

### 场景2：中型团队（10-50人）

**方案**：Vercel + PlanetScale
- 共享数据库
- 多用户协作
- 免费使用

**步骤**：
1. 创建 PlanetScale 数据库
2. 部署后端和前端
3. 配置环境变量

### 场景3：大型团队（>50人）

**方案**：云服务器
- 完全控制
- 高性能
- 数据安全

**步骤**：
1. 购买云服务器
2. 安装 Node.js 和 MySQL
3. 部署代码
4. 配置域名
